Functions


1. Perform centralized server management. You can register with and manage multiple servers.
2. Query basic server information, including the product model, IP address, device location, health status, power-on status, product serial number, asset tag, version, and CPU and memory brief information.
3. Query current health events of the server.
4. Query and set out-of-band network information.
5. Query hardware information.
6. Query server real-time status, including the temperature and health status of monitored components.
7. Query and set device location.
8. Query power consumption.
9. Query and set x86 boot device and mode.
10. Power on and off the server.
11. Set the UID indicator.
12. Query firmware information.
13. Collect hardware and software information, and share collected information in graphics or PDF files.
14. Perform one-click information collection and share the collected information.
